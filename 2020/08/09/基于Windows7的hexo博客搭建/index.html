基于Windows7的hexo博客搭建

基于windows7的hexo博客搭建

1.安装node.js
2.安装git 参考https://zhuanlan.zhihu.com/p/123195804
2.打开cmd，Windows+R，输入cmd进入命令行
3.查看安装的node版本及npm版本。
4.npm install -g cnpm –registry=http://registry.npm.taobao.org #安装淘宝的cnpm 管理器
5.cnpm install -g hexo-cli #安装hexo框架
6.hexo -v #查看hexo版本
7.chdir 查看当前路径
8.mkdir blog 当前目录创建blog文件夹
9.cd blog 进入blog文件夹
10.blog文件夹下 hexo init 初始化博客
11.hexo s 启动博客，本地4000端口启动，localhost：4000
12.hexo n 创建新的博客文章
13.根据提示查看刚刚创建的博客文章文件，并可打开文件进行写博文
14.hexo clean 清理
15.hexo g生成
16.hexo s重新启动查看刚刚写的文章
17.github 创建新的repository，“用户部署个人博客的github仓库的命名必须符合特定要求才行。”
18.cnpm install –save hexo-deployer-git #在blog目录下安装git部署插件
19.#配置_config.yml

Deployment

Docs: https://hexo.io/docs/deployment.html

deploy:
type: git
repo: https://github.com/YourGithubName/YourGithubName.github.io.git
branch: master

20.hexo d #部署到Github仓库里
21.git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia #下载yilia主题到本地。
22.#修改hexo根目录下的 _config.yml 文件 ： theme: yilia

23.hexo c #清理一下
hexo g #生成
hexo d #部署到远程Github仓库
https://YourGithubName.github.io/ #查看博客
